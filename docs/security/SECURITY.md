# Security Policy (Virdato)

## Reporting a vulnerability
Email: security@virdato.com (or your designated alias)
Include:
- affected component (contract name, API route, UI)
- steps to reproduce / PoC
- impact analysis
- chain + contract addresses (if onchain)

## Scope
In scope:
- smart contracts in `contracts/src`
- backend APIs in `backend/`
- Next.js app in `src/`
- CI/CD workflows

Out of scope:
- phishing/social engineering of team members
- third-party providers you don't control (unless misconfigured by Virdato)

## Safe Harbor
If you follow responsible disclosure and avoid harming users/funds,
we will not pursue legal action against you.

## Response targets (initial)
- acknowledge within 48 hours
- initial triage within 5 business days
